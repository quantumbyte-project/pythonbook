3.1 Condicionales#

Las estructuras de control nos brindan la capacidad de dirigir y controlar el flujo de nuestro código. Son especialmente útiles cuando no deseamos que todas las líneas de código se ejecuten en todas las situaciones, o cuando necesitamos repetir ciertos bloques de código varias veces. En esta sección, nos centraremos en los condicionales, los cuales son una de las estructuras de control más fundamentales.

Los condicionales son estructuras fundamentales que nos permiten tomar decisiones en un programa y seguir diferentes caminos de ejecución. A diferencia de la ejecución lineal de arriba hacia abajo, los condicionales nos brindan la capacidad de saltar o ejecutar ciertas partes de nuestro programa en función de condiciones específicas.

Imagina que tienes un programa con una serie de instrucciones. Normalmente, todas estas instrucciones se ejecutarían en orden. Sin embargo, con los condicionales, podemos establecer condiciones que determinan qué partes del programa se ejecutan y cuáles se omiten (veáse Fig.condicionales). Esto nos permite controlar el flujo de ejecución y adaptar el comportamiento del programa según diversas circunstancias.

Por ejemplo, podemos definir una condición que verifique si un número es mayor o menor que otro, y en función de eso, ejecutar diferentes bloques de código. También podemos establecer condiciones que activen o desactiven ciertas funcionalidades en nuestro programa.

Los condicionales nos brindan una poderosa herramienta para crear programas más dinámicos e interactivos. Con ellos, podemos implementar lógica de toma de decisiones y adaptar el comportamiento de nuestro programa en tiempo de ejecución.

3.1.1 Condicionales if y else#

Las estructuras de condicionales más simples y comunes son los bloques de código if y else.

El condicional if nos permite evaluar una condición y, si esta es verdadera, ejecutar un bloque de código específico. En otras palabras, nos permite tomar una decisión y ejecutar una acción si se cumple cierta condición.

Por otro lado, el condicional else nos brinda la posibilidad de especificar un bloque de código alternativo que se ejecutará si la condición evaluada en el if resulta ser falsa. Esto nos permite definir un comportamiento alternativo para situaciones en las que la condición no se cumpla.

En Python, la estructura if-else tiene la siguiente forma:

if condición:
    
    Instrucción

else:

    Instrucción

Estas estructuras condicionales nos permiten crear programas más flexibles y adaptativos, ya que podemos controlar qué partes del código se ejecutan en función de las condiciones que establezcamos. Además, podemos combinar múltiples bloques if y else para crear estructuras condicionales más complejas y abarcar diferentes escenarios de decisión en nuestros programas.

De manera similar, hay veces que como no queremos ejecutar cierto código si no se cumple algo si queremos ejecutar a otras cosas entonces para eso junto con el el if utilizamos lo que conocemos como el else, es decir, recordemos los códigos o los programas su flujo de ejecución son de arriba hacia abajo entonces vaya va de arriba hacia abajo, llega este a este condicional y dice: “bueno no se cumple la condición” y el else nos ayuda que si no se cumple esta condición se va a ser lo que esté dentro del else, es decir “si se cumple y si no”.

##Muliple cantidad de Condiciones.

Como vimos, en los condicionales podemos colocar una sola condición, pero ahora imaginemos que en un “if”, necesitamos que se cumplan dos condiciones o que al menos una de ellas sea válida para que se ejecute el bloque con instrucciones que contiene el if, a fuerza necesitamos que las dos sean validas o al menos una de ellas, en ese caso ¿que hacemos?, ¿usamos dos “if”?, aunque en principio la idea es válida, se podría colocar y es perfectamente válido, que pasará si queremos varias condiciones para un if, colocamos tantos if de una sola condición, pero al final, el programa quedaría con múltiples if indentados, y un código muy difícil de leer, ¿podemos colocar más de una condición en un mismo if para ahorrarnos todo esto? La respuesta es si, para esto utilizaremos “or” y “and”.

Imaginemos que queremos leer un valor numerico del 1 al 5, y dependiendo del valor queremos que imprima algo. Entonces tenemos 5 poscibles valores, el 1,2,3,4 y el 5, si quisieramos imprimir cada uno de ellos con lo que sabemos hasta ahora lo que podria ocurrirsele seria algo parecido a esto:

V=int(input("Ingrese un valor [del 1 al 5]: "))

if V==1:
  print("El valor es 1")
else:

  if V==2:
    print("El valor es 2")
  else:

    if V==3:
      print("El valor es de 3")
    else:

      if V==4:
        print("El valor es 4")
      else:
        if V==5:
          print("El valor es 5")
        else:
          print("Error ingrese nuevamente el valor") 

Ingrese un valor [del 1 al 5]: 3
El valor es de 3

Si ejecutamos el programa en efecto funciona, pero cada vez vamos a requerir mas y mas identacion, y en un punto deja de ser de leer tan facil, esto no solo es un problema estetico, un programa dificil de leer es un programa dificil de corregir y mantener a largo plazo, por eso es importante valernos de otro recurso que nos ofrece Python, otro condicional llamado “elif”.

Para entender como funciona este condicional, tratemos de corregir el codigo de arriba:

V=int(input("Ingrese un valor [del 1 al 5]: "))

if V==1:
  print("El valor es 1")
elif V==2:
  print("El valor es 2")
elif V==3:
  print("El valor es de 3")
elif V==4:
  print("El valor es 4")
elif V==5:
  print("El valor es 5")
else:
  print("Error ingrese nuevamente el valor") 

Ingrese un valor [del 1 al 5]: 6
Error ingrese nuevamente el valor

Como vemos, el texto ahora resulta más legible, está más simplificado, y sobre todo, no tenemos tantas iteraciones.

La estructura es simple, siempre vamos a iniciar con un “if” de toda la vida, una vez colocado su bloque de instrucción, la siguiente línea después del bloque estará al mismo nivel que el “if” original, y con la palabra “elif” seguido de su condición y dos puntos funcionará como un “if” y un “else” a la vez en el siguiente sentido:

El elif es un "si no se cumple lo anterior... intenta con esta condición"

elif condición : 
   bloque de instrucciones

así hasta que terminemos con todas las posibles condiciones. Para un correcto funcionamiento, deberemos colocar un “else” al final, este “else” significa:

sí ninguno de los casos anteriores fue verdadero, realiza esto...

Y así de simple es cómo funciona el elif.

Con esto hemos terminado lo que son los condicionales que maneja Python, en otros lenguajes, como c, existen más, como lo podrían ser el switch, pero en Python solo tenemos el “if”, el “else” y el “elif”.
